# Pythontree

This project has been created to facilitate folder management

* ### Installation ###

		pip3 install pythontree
		pip3 install pythontree --upgrade #upgrade
		

* ### Roots ###
		
		import pythontree
		Tree = pythontree.Tree('*** you can choose the path to start ***')
		Tree.roots()
		
* ### Clean ###

		import pythontree
		Tree = pythontree.Tree('*** you can choose the path to start ***')
		Tree.clean()
