# Pythontree

This project has been created to facilitate folder management

* ### Installation ###

		pip3 install pythontree
		pip3 install pythontree --upgrade #upgrade
		

* ### Roots ###

		Tree = Tree('*** you can choose the path to start ***')
		Tree.root()
		

* ### Clean ###

		Tree = Tree('*** you can choose the path to start ***')
		Tree.clean()
